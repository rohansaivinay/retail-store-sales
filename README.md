# 🛒 Retail Store Inventory Sales Prediction

A Machine Learning project that predicts **Units Sold** using retail inventory data. This project demonstrates the complete ML workflow, including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and business insights.

---

## 📌 Project Overview

Retail businesses need accurate sales predictions to optimize inventory and reduce stock shortages or overstocking. This project uses the **Random Forest Regressor** algorithm to predict the number of units sold based on inventory, pricing, discounts, seasonality, and other business factors.

---

## 🎯 Objectives

- Analyze retail inventory data
- Perform Exploratory Data Analysis (EDA)
- Preprocess and prepare the data
- Build a Machine Learning model
- Predict Units Sold
- Evaluate model performance
- Identify the most important sales-driving factors

---

## 📂 Dataset

The dataset contains information such as:

- Product ID
- Category
- Region
- Inventory Level
- Units Ordered
- Price
- Discount
- Weather Condition
- Seasonality
- Holiday/Promotion
- Date
- **Target Variable:** Units Sold

## conclusion
This project successfully developed a Random Forest Regression model to predict Units Sold using retail inventory data. Through data preprocessing, exploratory data analysis, and machine learning, the model identified key factors influencing sales, such as inventory level, price, discounts, and seasonal trends. The insights generated can help businesses optimize inventory management, improve demand forecasting, and support better pricing and stocking decisions. Overall, this project demonstrates the practical application of machine learning in solving real-world retail business problems.
